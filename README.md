# zbox
### Next generation airbox
- Embedded System: Pi3 B+
- PM2.5 sensor: G5
- Temperature & Humidity: SHT31
- CO2: Sense Air S8
- TVOC: SGP30 *
- Light: TCH34725
- Time: DS3231
- Display: epaper 2.7inch
- GPS: NEO6M *
